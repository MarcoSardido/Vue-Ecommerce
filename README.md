## Installation

Clone the project

```sh
  git clone https://github.com/MarcoSardido/Vue-Ecommerce.git
```

Go to the project directory

```bash
  cd VueEcommerce
```

Install dependencies

```sh
  npm install
```

Start the server

```bash
  npm run dev
```
